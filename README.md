# MuterGresik
MuterGresik adalah platform rekomendasi wisata di Kabupaten Gresik berbasis website yang dapat memberi saran wisata dan kuliner di Gresik berdasarkan mood. 
Website ini kami buat dengan HTML, CSS, dan Javascript murni tanpa menggunakan library atau framework apapun sesuai dengan persyaratan lomba Festika Arek AI Jatim 2025.
